# FIFOService

Sistema de controle de pedidos que utiliza a estrutura de dados Fila (Queue), ou seja, o primeiro pedido feito é o primeiro finalizado.

O contexto utilizado como exemplo neste sistema é de uma esfiharia fictícia chamada "Esfiha Express". O sistema comtempla:

- Cardápio de exemplo
- Registro de pedidos
- Ordenação de pedidos em lista
- Busca e printar um pedido único pelo seu ID
- Remoção dos pedidos prontos da fila
- Cancelamento de pedido por ID
